# Brutalist Constructivism – Locked AI Prompt System

This document is **canonical**.
All visual assets for the game MUST be generated using these prompts.
Deviations are errors, not creative alternatives.

---

## 0. Usage Rules (Non‑Negotiable)

- Prompts are **infrastructure**, not inspiration
- Always include:
  - GLOBAL STYLE PROMPT
  - GLOBAL NEGATIVE PROMPT
- Never add stylistic adjectives unless explicitly allowed
- If an asset violates the rejection checklist → discard immediately

---

## 8. ABSOLUTE REJECTION CHECKLIST

Discard the asset immediately if **ANY** apply:

- Rounded corners present
- Gradients or soft shadows present
- Friendly, playful, or expressive tone
- Painterly or illustrative style
- Excessive detail replacing silhouette
- Looks cinematic instead of operational
- Feels like a game UI instead of an ops room

---

## 9. Canon Statement

> This visual system defines authority, clarity, and control.
